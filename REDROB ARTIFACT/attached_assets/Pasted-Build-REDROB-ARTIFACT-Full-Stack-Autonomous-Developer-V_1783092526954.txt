Build REDROB ARTIFACT — Full Stack Autonomous Developer Verification Agent

Build a complete full-stack web app called REDROB ARTIFACT. This is an autonomous multi-agent AI system that verifies developer talent by cross-checking resume/skill claims against real GitHub activity and outputs a credibility score with evidence.
TECH STACK:

Frontend: React (port 3000)
Backend: Node.js + Express (port 5000)
AI: Gemini 2.0 Flash API for parsing and verification logic
Data: GitHub REST API (public endpoints, no auth needed)
Storage: In-memory cache (simple JS object) to avoid re-fetching same GitHub profile repeatedly

BACKEND (Node.js + Express):
Setup:

Enable CORS for http://localhost:3000
Parse JSON request bodies
Run on port 5000
Two endpoints:

GET /health — returns { status: "ok" }
POST /verify — main agent pipeline endpoint



Agent Pipeline (run these in sequence inside /verify):
Agent 1 — Intake/Parsing Agent:

Call Gemini 2.0 Flash API with the resume text
Prompt: "Extract all technical skills, frameworks, and languages from this resume text. Return ONLY a JSON array of skill names, nothing else. Example: ['Java', 'React', 'Node.js']"
Parse the returned JSON array of skills

Agent 2 — GitHub Retrieval Agent:

Extract username from input (strip https://github.com/ if present)
Call https://api.github.com/users/{username} — get profile info
Call https://api.github.com/users/{username}/repos?per_page=100 — get all public repos
From repos extract: repo names, languages, star count, fork count, pushed_at dates, sizes
Build a summary: total repos, unique languages used, top 3 most active repos by pushed_at

Agent 3 — Verification Agent:

For each skill from Agent 1, check if that language/framework appears in the GitHub repos fetched by Agent 2
Verdict logic:

VERIFIED: skill appears in 2+ repos with meaningful commit activity (size > 0, pushed in last 2 years)
PARTIAL: skill appears in 1 repo only
UNVERIFIED: skill not found in any repo


Call Gemini again with the skill verdicts + repo data to generate a human-readable evidence summary paragraph

Agent 4 — Scoring Agent:

Calculate score: (verified skills × 100 + partial skills × 50) / (total skills × 100) × 100
Round to nearest integer, cap at 100

Output — return this JSON:
json{
  "score": 78,
  "verdicts": {
    "Java": "verified",
    "React": "partial",
    "Python": "unverified"
  },
  "evidence": "Java strongly verified across 4 active repositories with recent commits. React found in 1 project. Python not found in any public repository.",
  "github_summary": {
    "username": "VARSHINI268",
    "total_repos": 12,
    "top_languages": ["Java", "JavaScript", "HTML"],
    "most_active_repo": "project-xyz",
    "profile_url": "https://github.com/VARSHINI268"
  }
}
Error handling:

If GitHub returns 404, return error: "GitHub profile not found. Check the username."
If Gemini API fails, return error: "AI verification failed. Try again."
If resume text is empty, return error: "Please paste your resume or skills list."
